# Install
*current release: `2017-11-06`*

This repo is managed with [GNU stow][stow]. **Backup** your dotfiles!

```bash
git clone git://github.com/twilton/dotfiles ~/.dotfiles
cd ~/.dotfiles
stow <application>
```
